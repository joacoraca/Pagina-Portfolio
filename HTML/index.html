Mis proyectos

NexLock

NexLock es un sistema inteligente de control de acceso desarrollado como proyecto de electrónica, diseñado para mejorar la seguridad mediante la combinación de hardware embebido y servicios en la nube. El sistema utiliza un ESP32 y un ESP32-CAM para gestionar la autenticación de usuarios a través de reconocimiento facial y un código PIN, permitiendo autorizar o denegar el acceso de forma automática.

La solución integra una cerradura motorizada, captura de imágenes en tiempo real y comunicación con un backend en Firebase, donde se realiza la verificación facial y se almacenan los registros de acceso. Además, incorpora sensores para detectar el estado de la puerta y ejecutar el cierre automático una vez que esta vuelve a cerrarse.

Con este proyecto adquirí experiencia en sistemas embebidos, programación de microcontroladores, integración con servicios en la nube, comunicación Wi-Fi, diseño de hardware y desarrollo de soluciones IoT orientadas a la seguridad.
