Proyecto Destacado

NexLock

Sistema Inteligente de Control de Acceso IoT

ESP32 ESP32-CAM Firebase Firestore JavaScript HTML5/CSS3

NexLock es un sistema inteligente de control de acceso físico que fusiona el desarrollo de software moderno en la nube con hardware embebido. Diseñado como proyecto de electrónica avanzada, ofrece una solución de seguridad robusta y en tiempo real.

¿Cómo funciona el ecosistema NexLock?

  • Hardware Embebido (ESP32 / ESP32-CAM): Gestiona la autenticación mediante lectura de código PIN y reconocimiento facial, y controla una cerradura motorizada con sensores que detectan el estado de la puerta para aplicar un cierre automático inteligente.
  • Base de Datos y Autenticación (Firebase): Utiliza Firebase para sincronizar en tiempo real el estado del cerrojo, almacenar imágenes de intentos de acceso sospechosos o autorizados, y guardar logs históricos en Firestore.
  • Panel Web Administrativo (Frontend): Un panel web construido en JavaScript puro donde los administradores pueden registrar usuarios, autorizar accesos remotamente, visualizar estadísticas en vivo (tasa de éxito de accesos, intentos fallidos) y monitorear logs en tiempo real.
